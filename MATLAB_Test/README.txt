INFO SCRIPT PER IL TEST DEL MODELLO DELLA RETE LENET5

Lo script MATLAB "script_test.m" permette di testare il comportamento della rete implementata in hardware analizzando un'immagine (a scelta dell'utente) presa dal datset MNIST-test.


Lo script genera anche un file binario della rispettiva immagine testata che può essere passato direttamente al testbench della rete completa copiando e incollando il file nella cartella ./VHDL/LeNet5/File_input.
