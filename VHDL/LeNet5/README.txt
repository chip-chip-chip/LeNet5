This folder contains many folders for VHDL files and a "compile.do" script for Modelism simuation.
In VHDL_common there is the top level VHDL block of LeNet5.
The final project is the result of a modular approach: each layer has been developed and tested by itself to be then included in the main entity. 

-- File_input  --> Inputs.txt files to feed the network;
-- File_output --> Outputs.txt files from the network;
-- File_matlab --> Matlab.txt files to compare results.
